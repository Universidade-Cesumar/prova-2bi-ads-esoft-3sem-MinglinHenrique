# Sistema de Almoxarifado

Este é um sistema simples de controle de almoxarifado desenvolvido com HTML, CSS e JavaScript, com o objetivo de gerenciar o cadastro, listagem e retirada de materiais.

## Funcionalidades

- Cadastro de materiais com nome e quantidade
- Listagem de materiais em tabela
- Controle de retirada de estoque
- Atualização dinâmica dos dados na interface

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Estrutura do projeto

- index.html -> Estrutura principal da aplicação
- style.css -> Estilização da interface
- main.js -> Lógica do sistema

## Como executar

- Baixe ou clone este repositório
- Abra o arquivo index.html em qualquer navegador moderno
- Utilize a interface para cadastrar e gerenciar materiais
